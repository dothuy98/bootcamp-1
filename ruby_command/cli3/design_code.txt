fetch minimum priceツール

目的 :
自分が欲しい商品の最安値を取得する

コア機能 :
サイトにいって検索して最安値で並び替えをするという一連の工程をいちいち何度も行うのは面倒なため、
それをスクレイピングで自動化する。
出力は最安値の商品を表示するページのURLとする。

スクレイピング対象とするサイト : メルカリ

追加機能 :
・複数のサイトでの比較を対応する。

・対象サイトに楽天市場、yahooショッピング、ラクマを追加する。

・商品に大きな欠陥がある場合を踏まえて上位何位までを表示するかをオプションで指定する。

・not条件を加える。
例えばある本が欲しいが裁断済みなどの本は使い勝手が悪いので検索から除外したいなどの検索方法を実現する。

・標準価格を出力するために価格の平均を算出する。
