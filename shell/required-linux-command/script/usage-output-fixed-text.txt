使い方

  sh oft.sh <登録したファイル> (<オプション> ...) 

定型文の登録、登録した定型文の出力を行えます。

"xxx"で入力した部分は、定型文を呼び出すたびに変更することができ、柔軟な定型文の作成ができます。
また、"xxx"は、-r <文字列>で変更できます。

登録
  sh oft.sh -f <登録したいファイルの名前>

出力
  sh oft.sh <すでに登録済みの出力したいファイル名>
   ※登録されていないファイルの場合、そのファイル名で登録しようとします。

登録ファイルの一覧表示
  ファイル名 : sh oft.sh -v
  ファイルのテキストも表示 : sh oft.sh -t

オプションについて
-r <文字列> : デフォルトでは"xxx"の置換文字を別の文字列に変更します。
             ファイルの新規登録の際に使用します。
-f <file> : ファイルで新規登録を行う。
-v : 登録した定型文のファイル名の名前を表示。
-t : 登録した定型文の名前とテキストを表示する
